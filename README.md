# 🌤️  Weather 

A modern weather dashboard built with Django that provides real-time weather information and live world clocks.

## 🚀 Features

- 🔍 Search weather by city
- 🌡️ Real-time temperature
- ☁️ Weather condition
- 💧 Humidity
- 🌬️ Wind speed
- 🕒 Local time
- 🌍 Live World Clock
  - India
  - USA
  - London
  - Japan
  - Sydney
- 🌙 Dark / Light Mode
- 📱 Responsive Design
- ✨ Smooth Animations

---

## 🛠️ Technologies Used

- Python
- Django
- HTML5
- CSS3
- JavaScript
- Font Awesome
- Weather API

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/ghogharifenil/Weather.git

```

### Create Virtual Environment

```bash
python -m venv env
```

### Activate Environment

Windows:

```bash
env\Scripts\activate
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run Server

```bash
python manage.py runserver
```

Open:

```text
http://127.0.0.1:8000/
```
